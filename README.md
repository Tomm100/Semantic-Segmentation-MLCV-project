# GAN-Augmented Chest X-Ray Classification

## Project Proposal — Machine Learning for Computer Vision

### Objective

The goal of this project is to investigate the effectiveness of **GAN-based data augmentation** for improving the classification of chest X-ray images into two categories: **Normal** and **Pneumonia**. Medical imaging datasets often suffer from class imbalance and limited availability of labeled samples; this project addresses both issues by generating synthetic training images using a conditional Deep Convolutional GAN (cDCGAN) and evaluating their impact on a downstream classifier.

### Dataset

- **Source**: [Chest X-Ray Images (Pneumonia)](https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia)
- **Content**: 5,863 JPEG chest X-ray images (anterior-posterior), organized into train, validation, and test splits
- **Classes**: NORMAL (1,341 train) and PNEUMONIA (3,875 train) — naturally imbalanced (~1:3 ratio)
- **Subset strategy**: A reduced 25% subset is used for training the GAN (335 Normal, 970 Pneumonia), simulating a realistic low-data medical scenario

### Architecture

| Component | Architecture | Details |
|---|---|---|
| **Classifier** | ResNet-18 (pre-trained on ImageNet) | Final fully-connected layer replaced with a binary classification head |
| **Generator** | cDCGAN Generator (d=128) | Conditional on class label; 5 transposed conv layers; outputs 64×64 grayscale images |
| **Discriminator** | cDCGAN Discriminator (d=32) | Conditional on class label; 5 conv layers with dropout regularization |

### Training Setup

#### Phase 1 — Classifier Baseline
- **Method**: Fine-tuning of the pre-trained ResNet-18 on the reduced (25%) training subset
- **Purpose**: Establish baseline performance on the imbalanced, low-data setting

#### Phase 2 — cDCGAN Training
- **Method**: Training the cDCGAN from scratch on the same 25% subset
- **Key techniques** to stabilize GAN training on medical images:
  - Asymmetric learning rates (G: 0.0002, D: 0.00005)
  - Label smoothing (real: 0.9, fake: 0.1)
  - Instance noise with linear annealing
  - Dropout regularization in the discriminator
  - 2× generator updates per discriminator update
- **Epochs**: 100 with learning rate decay at epochs 60 and 80
- **Output**: 500 synthetic images per class

#### Phase 3 — Augmented Classifier
- **Method**: Fine-tuning ResNet-18 on the original subset + synthetic images generated by the cDCGAN
- **Purpose**: Evaluate whether GAN-generated data improves classification, especially for the minority class (NORMAL)

### Evaluation Metrics

| Metric | Purpose |
|---|---|
| **Accuracy** | Overall correctness |
| **Precision** | Reliability of positive predictions (reduce false alarms) |
| **Recall** | Sensitivity — critical in medical diagnosis (minimize missed pneumonia cases) |
| **F1-Score** | Harmonic mean of precision and recall |
| **Confusion Matrix** | Detailed breakdown of classification errors per class |

The primary comparison is **Phase 1 (baseline) vs Phase 3 (augmented)**, with particular attention to whether synthetic data improves recall on the minority class and overall F1-score.
